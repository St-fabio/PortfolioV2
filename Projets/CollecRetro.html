Gestionnaire de collection de jeux rétro

Description :

Projet créé pour appliquer les base de java vu en cours (Listes, fonction, procédure, etc...), ce projet a rapidement transormé mes vacances de la toussaint en semaine de 35h de développement.
Le programme n'est pour l'instant pas équipé d'interfaces graphiques.
Ce projet est née de l'idée qu'il n'existe pas encore d'outils libre et gratuit permettant de connaitre le prix d'une collection de jeux vidéo.
Le programme va bientôt êtres totalement recommencer pour avoir de nouvelle fonctionnalité et une interface graphique.
